## Function: `{event}`

```
{event name="string"}
```

Fire an event with the specified `name`. [Learn more about events in Vanilla.](../../../developers/plugins/#events-%26-handlers)

### Parameters

Parameter   | Type      | Default   | Description
---         | ---       | ---       | ---
__`name`__  | `string`  | none      | Name of the event to fire
