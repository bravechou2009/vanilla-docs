## Function: `{pocket}`

```
{pocket name="string"}
```

Writes a pocket to the page. Pockets placed in the template in this way should have their location set to 'custom'.

### Parameters

Parameter       | Type      | Default   | Description
---             | ---       | ---       | ---
__`name`__      | `string`  | none      | The name of the pocket
